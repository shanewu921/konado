# 静态诊断规则

## 规则 1：KS-DIAG-001 — 跳转到未定义的 branch 标签

| 属性 | 值 |
|------|-----|
| **ID** | `KS-DIAG-001` |
| **严重级别** | Error |
| **触发模式** | `choice "..." -> <branch_id>` 或 `jump_branch <branch_id>` 引用的 `<branch_id>` 在脚本中未通过 `branch <branch_id>` 声明 |
| **错误消息** | `跳转标签 '<branch_id>' 不存在（当前已定义标签：<list>）` |
| **修复建议** | 确保目标 `branch <branch_id>` 已在脚本中定义，且名称大小写完全一致；检查拼写错误 |

## 规则 2：KS-DIAG-002 — 未声明角色即使用

| 属性 | 值 |
|------|-----|
| **ID** | `KS-DIAG-002` |
| **严重级别** | Warning |
| **触发模式** | 在 `"角色ID" "..."` 对话中使用未通过 `actor show` 声明的角色ID |
| **错误消息** | `角色 '<角色ID>' 未通过 actor show 声明` |
| **修复建议** | 在对话前添加 `actor show <角色ID> <状态> at <位置>` |

## 规则 3：KS-DIAG-003 — 资源路径/标识符可能无效

| 属性 | 值 |
|------|-----|
| **ID** | `KS-DIAG-003` |
| **严重级别** | Warning |
| **触发模式** | `background <name>` / `play bgm <name>` / `play sfx <name>` 中的资源名包含特殊字符（空格、`/`、`\`）或长度超过 128 字符 |
| **错误消息** | `资源名 '<name>' 包含特殊字符，请使用纯字母数字下划线` |
| **修复建议** | 将资源名改为 `[a-zA-Z0-9_]+` 格式，特殊字符替换为下划线 |

## 规则 4：KS-DIAG-004 — 选项块结构错误

| 属性 | 值 |
|------|-----|
| **ID** | `KS-DIAG-004` |
| **严重级别** | Error |
| **触发模式** | `choice "..." -> <id>` 后缺少 `->` 运算符或缺少目标分支名；或 `choice` 行中选项文本未用双引号包裹 |
| **错误消息** | `choice 语句结构错误：缺少 '->' 或目标分支名` |
| **修复建议** | 确保格式为 `choice "选项文本" -> 分支名称`，文本用英文双引号包裹，`->` 两侧有空格 |

## 规则 5：KS-DIAG-005 — 变量未初始化

| 属性 | 值 |
|------|-----|
| **ID** | `KS-DIAG-005` |
| **严重级别** | Warning |
| **触发模式** | 在 `if %var op val:` 条件判断中使用了未通过 `set` 初始化的持久变量 `%var` |
| **错误消息** | `变量 '%var' 未在脚本中初始化，条件判断可能始终为假` |
| **修复建议** | 在条件判断前添加 `set %var = <初始值>`；或确认变量已在编辑器的 `KND_VariableStore` 检查器中预设 |

## 规则 6：KS-DIAG-006 — branch 嵌套

| 属性 | 值 |
|------|-----|
| **ID** | `KS-DIAG-006` |
| **严重级别** | Error |
| **触发模式** | 在 `branch <id>` 块内部又声明了另一个 `branch <id2>` |
| **错误消息** | `branch 内不能嵌套 branch，请使用平铺结构` |
| **修复建议** | 将嵌套的 `branch` 移至顶层，通过 `jump_branch` 实现跳转 |

## 规则 7：KS-DIAG-007 — if 嵌套

| 属性 | 值 |
|------|-----|
| **ID** | `KS-DIAG-007` |
| **严重级别** | Error |
| **触发模式** | 在 `if ... endif` 块内部又出现了 `if` 语句 |
| **错误消息** | `条件判断不支持嵌套，请使用平铺的 if/endif 结构` |
| **修复建议** | 将嵌套条件改为多个独立的平铺 `if/endif` 块 |

## 规则 8：KS-DIAG-008 — 缩进错误

| 属性 | 值 |
|------|-----|
| **ID** | `KS-DIAG-008` |
| **严重级别** | Error |
| **触发模式** | `branch` 块内的语句没有缩进，或缩进层级不一致（应为 4 空格或 1 Tab） |
| **错误消息** | `branch 块内语句缩进错误，应使用 4 空格或 1 个 Tab 缩进` |
| **修复建议** | 确保 `branch` 块内所有语句至少缩进 1 级（4 空格或 1 Tab） |

## 规则 9：KS-DIAG-009 — 重复创建演员

| 属性 | 值 |
|------|-----|
| **ID** | `KS-DIAG-009` |
| **严重级别** | Error |
| **触发模式** | 对同一角色ID多次调用 `actor show` 而没有显式 `actor exit` |
| **错误消息** | `角色 '<名称>' 已存在，请检查是否重复创建` |
| **修复建议** | 再次显示前先调用 `actor exit <名称>`；或使用 `actor change` 切换状态 |

## 规则 10：KS-DIAG-010 — end 后存在无效代码

| 属性 | 值 |
|------|-----|
| **ID** | `KS-DIAG-010` |
| **严重级别** | Warning |
| **触发模式** | `end` 语句后还有非注释、非空行的内容 |
| **错误消息** | `end 后的代码不会被执行，请移除或移到 end 之前` |
| **修复建议** | 删除 `end` 之后的所有语句，或将需要的语句移到 `end` 之前 |
