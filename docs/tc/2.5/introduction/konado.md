# 關於 Konado

## 專案簡介

Konado 是一款基於 Godot Engine 的視覺小說引擎框架，由開源社群驅動。我們致力於為開發者提供高效、便捷的視覺小說開發體驗，幫助創作者快速構建高品質的視覺小說作品，為創作賦能。我們追求更友好、更高效的工具，讓遊戲創作者能夠專注於故事敘述與藝術表現，而不必陷入程式實作的細節之中。

## 專案榮譽

- Konado 已被收錄至 Godot Engine 官方 [awesome-godot](https://github.com/godotengine/awesome-godot) 專案的「Plugins and scripts」部分，作為推薦插件之一。
- Konado 榮幸獲得 [2025 AtomGit 百大開源專案](https://atomgit.com/GitCode/G-Star100) 榮譽，並被收錄於 [AtomGit G-Star Landscape 3.0](https://gstar-landscape.gitcode.com/?item=音视频-图形-游戏-3d--游戏引擎-unity-unreal-godot--konado) 生態全景圖中。
