# 周邊 IP 介紹

## Kona 介紹

Kona 是 Konado 的專屬看板娘，以**活潑可愛、充滿元氣**的少女形象為核心。她不僅是品牌視覺的重要符號，也成為使用者與 Konado 之間親和而自然的互動載體。

## Kona 形象

Kona 的身影貫穿 Konado 的各個場景。無論是啟動畫面、官方網站首頁，還是產品文件，都能透過她的形象為品牌注入鮮活感。隨著 Konado 的版本迭代，Kona 的形象也會同步進化，既貼合新版本的設計風格，也在細節打磨中變得更精緻、更具辨識度。

通常在 Konado 的**大版本更新**時，Kona 會推出專屬的全新形象。每一款形象都有獨特的設計巧思，既延續她的核心氣質，也為產品增添更多趣味與記憶點。

當然，在小版本更新時，Kona 也會有一些小變化，例如換上不同季節的服飾，或增加一些小配件，讓形象更加豐富多元。
