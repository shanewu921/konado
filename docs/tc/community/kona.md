# 週邊 IP 介紹

## Kona 介紹
Kona 是 Konado 的專屬看板娘，以**活潑可愛、充滿元氣**的少女形象為核心，不僅是品牌視覺的重要符號，更成為使用者與 Konado 之間具親和力的互動載體。


## Kona 形象
Kona 的身影貫穿於 Konado 的全場景中，無論是啟動介面、官網首頁，還是產品文件，都能透過她的形象為品牌注入鮮活感。隨著 Konado 的版本迭代，Kona 的形象也會同步進化——既貼合新版本的設計風格，也在細節打磨中變得更精緻、更具辨識度。

通常在 Konado 的**大版本更新**時，Kona 會推出專屬的全新形象。每一款形象都有獨特的設計巧思，既延續了她的核心氣質，也為產品增添了更多趣味與記憶點。

當然在小版本更新時，Kona 也會有一些小變化，比如換上不同季節的服飾，或是增加一些小配件，讓形象更加豐富多元。
