# 建立演員

## 功能描述

在對話場景中建立一個演員。

## 語法結構

```text
actor show [角色ID] [狀態] at [水平座標]
```

## 參數詳解
| 參數 | 必需 | 範例 | 說明 |
|------|------|------|------|
| 角色ID | 是 | `alice` | 角色資源識別符 |
| 狀態 | 是 | `angry` | 立繪狀態 |
| 橫向座標 | 是 | `2` | 水平位置，表示位於幾等分位置 |

## 範例
```text
# 顯示角色
actor show alice normal at 2
```
