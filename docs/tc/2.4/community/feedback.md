# 建議與回饋

如果您對 `Konado` 專案的文件、開發流程、具體程式碼等內容有任何建議，或發現文件與程式碼中的錯誤，都歡迎向我們回饋。

如果您願意以任何方式參與，或為我們提供協助，請[聯絡我們](#建議管道)。作為一個社群協作的開源專案，您的貢獻會讓 `Konado` 變得更好。

## 問題回饋

請向官方倉庫[提交 issue](https://github.com/godothub/konado/issues)，主要分為以下三種類型：

- **Bug 回報**：描述問題，需包含重現步驟
- **功能建議**：在 Issue 區描述需求背景與預期效果
- **文件改進**：在 Issue 區描述需要改進的地方

## 建議管道

* [提交 issue](https://github.com/godothub/konado/issues)（優先考慮，方便我們追蹤問題）

* [加入群組回饋](https://pd.qq.com/g/godot)

* [發文求助](https://pd.qq.com/g/GodotHub999)
