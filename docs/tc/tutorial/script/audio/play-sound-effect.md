# 播放音效

## 功能概述
用於在特定場景或互動節點中播放指定的音效檔案，增強場景氛圍和互動回饋。播放音效是即時執行的，不會阻塞後續腳本執行。

## 語法規範
```
play sfx <音效名稱>
```

## 參數說明
| 參數       | 是否必填 | 範例         | 說明                     |
|------------|----------|--------------|--------------------------|
| 音效名稱   | 是       | door_open    | 需播放的音效檔案識別名稱 |

## 程式碼範例
```
play sfx door_open
play sfx button_click
play sfx notification
```
