# 播放 BGM

## 功能概述
用於在特定場景或互動節點中播放指定的背景音樂（BGM），用於營造場景氛圍和情緒渲染。播放 BGM 是即時執行的，不會阻塞後續腳本執行。

## 語法規範
```
play bgm <音樂名稱>
```

## 參數說明
| 參數       | 是否必需 | 範例         | 說明                     |
|------------|----------|--------------|--------------------------|
| 音樂名稱   | 是       | main_theme   | 需播放的 BGM 檔案識別名稱    |

## 程式碼範例
```
play bgm main_theme
play bgm peaceful_morning
play bgm tension_theme
```
