# 選項跳轉

## 功能概述

此功能需與**分支功能**搭配使用，核心作用是在互動選項標籤中配置跳轉邏輯，點擊對應選項後可直接跳轉到指定的分支節點。每個 `choice` 語句會建立一個選項，多個相鄰的 `choice` 行會合併為一個選項組，並在互動時同時展示。

## 語法規範

```text
choice "選項內容" -> 分支名稱
```

| 參數 | 必填 | 範例 | 說明 |
|------|------|------|------|
| 選項內容 | 是 | `"選擇咖啡"` | 互動時展示的選項文字內容 |
| 分支名稱 | 是 | `coffee_choice` | 需跳轉的目標分支唯一識別碼 |

## 使用範例

每個選項單獨一行：

```text
choice "Green tea" -> green_tea
choice "Black tea" -> black_tea
```

也可以在一行中連續書寫多個選項：

```text
choice "Green tea" -> green_tea "Black tea" -> black_tea
```

## 注意事項

1. 分支名稱需與專案中定義的分支識別碼完全一致，區分大小寫，且不可包含空格、特殊符號；
2. 選項內容需使用英文雙引號包裹，支援一般文字字元；
3. `->` 為選項與分支名稱之間的分隔符號，不可省略。
