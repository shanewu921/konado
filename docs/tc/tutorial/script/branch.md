# 分支

## 功能描述

分支是用於組織腳本內容的一種方式，透過分支可以方便地管理和調用腳本中的不同部分，用於實現複雜的跳轉邏輯，該功能一般搭配選項使用。

可以將標籤理解為腳本中的「書籤」，透過標籤可以快速定位到腳本中的特定位置，由標籤和縮排包裹的內容即為標籤的內容，這部分腳本內容在執行當前標籤對話的時候，會被添加到對話列表中，否則只會保存到標籤對話中，不會觸發播放。

分支不可以巢狀使用，分支的縮排層級必須和對話的縮排層級一致，否則會導致無法正常識別，以下是錯誤的範例：

```text
# 錯誤的範例
branch drink_water
    "kona" "我想喝水"
        branch eat_cake
            "kona" "我想吃蛋糕"
branch drink_tea
    "kona" "我想喝茶"
```


## 語法結構
```text
branch [標籤ID]
    [腳本內容]
```

## 參數詳解
| 參數 | 必填 | 範例 | 說明 |
|------|------|------|------|
| 標籤ID | 是 | `drink_water` | 標籤識別碼 |


## 範例

```text
branch drink_water
    "kona" "我想喝水"
branch drink_tea
    "kona" "我想喝茶"
```
