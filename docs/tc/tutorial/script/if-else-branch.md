# If Else 分支

## 功能描述

根據數值條件判斷，選擇播放對應的對話內容，如果條件不滿足，則播放 else 中的對話內容，如果沒有 else，則不播放任何對話內容。
%變數為自定義變數，值為整數數值，可在對話管理中設定並透過「%變數名」的方式引用，`==` 為判斷條件，表示等於。

## 語法結構
```
if %變數 == 值:
    <對話>
    <對話>
else:
    <對話>
endif
```


## 範例

```
if %love == 0:
    "Kona" "今天想喝點什麼？"
else:
    "Kona" "又見面了！"
    "Kona" "今天想喝點什麼？"
endif
```
