# Installation

## Dependencies

1. Konado plugin installed (required)
2. Godot version with C# support (recommended 4.5+)

## Installation Steps

1. Extract the konadotnet plugin into your Godot project's `addons` directory
2. Build the project and ensure there are no errors
3. In the Godot editor, go to `Project -> Project Settings -> Plugins`, then enable both the Konado and Konadotnet plugins
