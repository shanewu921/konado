# Install Konado WebTool

## Basic Dependencies

1. Godot 4.0+
2. Web platform target

## Installation Steps

1. Copy the `konado_webtool` plugin to the `addons` directory of your Godot project
2. In the Godot editor, go to `Project -> Project Settings -> Plugins` and enable the KonadoWebTool plugin
