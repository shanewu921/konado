# Actor Exit

## Description
Make the specified actor leave the current scene.

## Syntax
```text
actor exit [character ID]
```

## Parameters
| Parameter | Required | Example | Description |
|------|------|------|------|
| Character ID | Yes | `alice` | Character resource identifier |

## Example

```text
# Character exits the scene
actor exit alice
```
