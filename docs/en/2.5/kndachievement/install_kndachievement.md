# Installation

## Installation Steps

1. Copy the `KonadoAchievement` plugin to the `addons` directory of your Godot project
2. In the Godot editor, go to `Project -> Project Settings -> Plugins` and enable the `KonadoAchievement` plugin

## Autoload

KND_AchievementManager adds KND_AchievementManager under the project's `root` node automatically. Manual loading is not required.
