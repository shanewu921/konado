# About Konado

## Project Overview

Konado is a visual novel engine framework based on Godot Engine and driven by the open-source community. We are committed to providing developers with an efficient and convenient visual novel development experience, helping creators quickly build high-quality visual novel works and empowering creative production. We pursue friendlier and more efficient tools so game creators can focus on storytelling and artistic expression instead of getting trapped in implementation details.

## Project Honors

- Konado has been included in the "Plugins and scripts" section of Godot Engine's official [awesome-godot](https://github.com/godotengine/awesome-godot) project as one of the recommended plugins.
- Konado was honored as one of the [2025 AtomGit Top 100 Open Source Projects](https://atomgit.com/GitCode/G-Star100) and has been included in the [AtomGit G-Star Landscape 3.0](https://gstar-landscape.gitcode.com/?item=音视频-图形-游戏-3d--游戏引擎-unity-unreal-godot--konado) ecosystem map.
