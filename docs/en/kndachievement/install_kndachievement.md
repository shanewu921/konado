# Installation

## Installation Steps

1. Copy the `KonadoAchievement` plugin to your Godot project's `addons` directory
2. In the Godot editor, go to `Project -> Project Settings -> Plugins` and enable the `KonadoAchievement` plugin

## Autoload

`KND_AchievementManager` will be added to the project's `root` node automatically — no manual loading is required.
