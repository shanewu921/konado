# 创建演员

## 功能描述

在对话场景中创建一个演员

## 语法结构

```text
actor show [角色ID] [状态] at [水平坐标]
```

## 参数详解
| 参数 | 必需 | 示例 | 说明 |
|------|------|------|------|
| 角色ID | 是 | `alice` | 角色资源标识符 |
| 状态 | 是 | `angry` | 立绘状态 |
| 横向坐标 | 是 | `2` | 水平位置，几分之几 |



## 示例
```text
# 显示角色
actor show alice normal at 2
```
