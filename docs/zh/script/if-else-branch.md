# If Else 分支

## 功能描述

根据数值条件判断，选择播放对应的对话内容，如果条件不满足，则播放 else 中的对话内容，如果没有else，则不播放任何对话内容。
%变量为自定义变量，值为整数数值，可在对话管理中设置并通过“%变量名”的方式引用，`==` 为判断条件，表示等于。

## 语法结构
```
if %变量 == 值:
    <对话>
    <对话>
else:
    <对话>
endif
```


## 示例

```
if %love == 0:
    "Kona" "今天想喝点什么？"
else:
    "Kona" "又见面了！"
    "Kona" "今天想喝点什么？"
endif
```
