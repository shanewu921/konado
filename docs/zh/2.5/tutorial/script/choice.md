# 选项跳转
## 功能概述
该功能需与**分支功能**搭配使用，核心作用是在交互选项标签中配置跳转逻辑，点击对应选项后可直接跳转到指定的分支节点。每个 `choice` 语句创建一个选项，多个相邻的 `choice` 行会被合并为一个选项组，在交互时会同时展示。

## 语法规范

```
choice "选项内容" -> 分支名称
```

| 参数       | 必填 | 示例            | 说明                     |
|------------|------|-----------------|--------------------------|
| 选项内容   | 是   | "选择咖啡"       | 交互时展示的选项文本内容 |
| 分支名称   | 是   | coffee_choice   | 需跳转的目标分支唯一标识 |

## 使用示例

每个选项单独一行：

```
choice "Green tea" -> green_tea
choice "Black tea" -> black_tea
```

也可以在一行中连续书写多个选项：

```
choice "Green tea" -> green_tea "Black tea" -> black_tea
```

## 注意事项
1. 分支名称需与项目中定义的分支标识完全一致，区分大小写，且不可包含空格、特殊符号；
2. 选项内容需用英文双引号包裹，支持常规的文本字符；
3. `->` 为选项与分支名称之间的分隔符，不可省略；
