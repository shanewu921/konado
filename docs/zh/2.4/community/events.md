# 活动

Konado Project 会定期组织一些活动，也会参与到一些社区活动中。欢迎所有人参与。

## 2026 Konado 视觉小说创意周（K26）

### 主办方

- GodotHub
- Konado Project

### 活动介绍

2026 Konado 视觉小说创意周（K26），以 Godot + Konado 开源生态为基石，主打低门槛、高自由、重创意。也许是你的第一个视觉小说，也可能是你踏入原创叙事创作、独立游戏开发的第一步。我们期待每一份热爱与创意，在这里落地生根，被看见、被收录、被欣赏。
