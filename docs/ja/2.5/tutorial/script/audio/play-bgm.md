# BGM の再生

## 機能概要
特定のシーンやインタラクションノードで指定した背景音楽（BGM）を再生します。シーンの雰囲気や感情表現を作るために使用します。BGM の再生は即時に実行され、後続のスクリプト実行をブロックしません。

## 構文
```
play bgm <音楽名>
```

## パラメーター
| パラメーター | 必須 | 例 | 説明 |
|------------|----------|--------------|--------------------------|
| 音楽名 | はい | main_theme | 再生する BGM ファイルの識別名 |

## コード例
```
play bgm main_theme
play bgm peaceful_morning
play bgm tension_theme
```
