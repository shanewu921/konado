# 背景切り替え

## 機能説明
ゲームシーンの背景画像を切り替えます。トランジション効果にも対応しています。

## 構文
```text
background [画像リソース名] <効果タイプ>
```

## パラメーター
| パラメーター | 必須 | 例 | 説明 |
|------|------|--------|------|
| 画像リソース名 | はい | `morning_forest` | 拡張子を含まないテクスチャファイル名 |
| 効果タイプ | いいえ | `fade` | トランジション効果（デフォルト：即時切り替え） |

### 対応する効果タイプ

以下の背景切り替え効果に対応しています。各効果には固有のビジュアル表現があります。

| 効果 | 説明 |
|------|------|
| `none` | 即時切り替え |
| `fade` | フェードイン/アウト |
| `erase` | 消去 |
| `blinds` | ブラインド |
| `wave` | 波 |
| `vortex` | 渦 |
| `windmill` | 風車 |
| `cyberglitch` | サイバーグリッチ |

効果タイプを指定しない場合、デフォルトで `none`（即時切り替え）が使用されます。

## 例
```text
# 昼から夜へ切り替え（フェード効果）
background night_street fade

# 戦闘シーン切り替え（即時切り替え）
background battle_field none

# 回想シーン（消去効果）
background memory_flash erase

# 夢のシーン（渦効果）
background dream vortex
```
