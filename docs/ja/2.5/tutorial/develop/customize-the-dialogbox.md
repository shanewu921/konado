# 会話インターフェースのカスタマイズ

## 紹介

作品にカスタム会話インターフェースが必要な場合は、テンプレートの会話インターフェースを変更して実現できます。また、Godot Engine には強力なテーマシステムがあるため、カスタムテーマを変更して目的の効果を得ることもできます。

ただし、テンプレートの会話インターフェースを変更した後は Konado プラグインを更新しないでください。更新すると変更内容が上書きされます。

## テンプレートファイルの編集

`res://addons/konado/template/` は会話インターフェースのシーンです。このファイルを変更することで会話インターフェースをカスタマイズできます。

通常はノード上のスクリプトを変更せず、ノードのプロパティを変更してカスタマイズすることを推奨します。
