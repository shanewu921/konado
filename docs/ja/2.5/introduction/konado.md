# Konado について

## プロジェクト概要

Konado は Godot Engine をベースにした、オープンソースコミュニティ主導のビジュアルノベルエンジンフレームワークです。私たちは、開発者に効率的で扱いやすいビジュアルノベル開発体験を提供し、クリエイターが高品質なビジュアルノベル作品を素早く構築できるよう支援することを目指しています。より親しみやすく効率的なツールを追求し、ゲーム制作者が実装の細部に悩まされることなく、物語の表現とアートに集中できる環境を作ります。

## プロジェクトの受賞・掲載

- Konado は Godot Engine 公式の [awesome-godot](https://github.com/godotengine/awesome-godot) プロジェクトの「Plugins and scripts」セクションに、推奨プラグインの一つとして掲載されています。
- Konado は [2025 AtomGit Top 100 Open Source Projects](https://atomgit.com/GitCode/G-Star100) に選出され、[AtomGit G-Star Landscape 3.0](https://gstar-landscape.gitcode.com/?item=音视频-图形-游戏-3d--游戏引擎-unity-unreal-godot--konado) のエコシステムマップにも収録されています。
