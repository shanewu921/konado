# 提案とフィードバック

`Konado` プロジェクトのドキュメント、開発プロセス、具体的なコードなどについて提案がある場合、またはドキュメントやコードの誤りを見つけた場合は、ぜひフィードバックをお寄せください。

何らかの形で参加したい場合や支援したい場合は、[お問い合わせ](#フィードバック窓口)ください。コミュニティ協作型のオープンソースプロジェクトとして、皆さんの貢献は `Konado` をより良いものにします。

## 問題の報告

公式リポジトリに [issue を作成](https://github.com/godothub/konado/issues)してください。主に以下の 3 種類があります。

- **Bug 報告**：問題を説明し、再現手順を含めてください
- **機能提案**：Issue で要望の背景と期待する動作を説明してください
- **ドキュメント改善**：改善が必要な箇所を Issue で説明してください

## フィードバック窓口

* [issue を作成](https://github.com/godothub/konado/issues)（問題を追跡しやすいため優先）

* [グループに参加してフィードバック](https://pd.qq.com/g/godot)

* [投稿して質問](https://pd.qq.com/g/GodotHub999)
