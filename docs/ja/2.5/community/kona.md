# 関連 IP 紹介

## Kona について

Kona は Konado 専属のマスコットです。**元気でかわいらしく、活発な**少女像を核としており、ブランドビジュアルの重要なシンボルであるだけでなく、ユーザーと Konado を親しみやすくつなぐ存在でもあります。

## Kona のビジュアル

Kona は Konado のあらゆる場面に登場します。起動画面、公式サイトのトップページ、製品ドキュメントなど、さまざまな場所でブランドに生き生きとした印象を与えます。Konado のバージョン更新に合わせて Kona の姿も進化し、新バージョンのデザインスタイルに寄り添いながら、細部の作り込みによってより洗練され、識別しやすいデザインになっていきます。

通常、Konado の**メジャーバージョン更新**では、Kona の新しい専用ビジュアルが公開されます。どのビジュアルにも独自の工夫があり、彼女の核となる魅力を受け継ぎつつ、製品にさらなる楽しさと記憶に残る要素を加えます。

もちろん、マイナーバージョン更新でも、季節ごとの衣装や小物の追加など、Kona に小さな変化が加わることがあります。これにより、表現はさらに豊かで多彩になります。
