# Konado .NET API

Konado .NET API は Konado の C# 拡張レイヤーです。

主な機能:

- `KonadoAPI` 自動読み込み入口
- `DialogueManagerAPI` による会話マネージャー制御
- Konado 会話フローイベントの購読
- `.ks` スクリプト解析
- `KND_Dialogue`、`KND_Shot`、`KND_DialogueChoice` などの GDScript リソースラッパー

まずインストールガイドを読み、その後 API リファレンスを確認してください。
