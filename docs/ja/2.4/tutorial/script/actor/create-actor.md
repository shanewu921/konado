# アクターの作成

## 機能説明

会話シーンにアクターを作成します。

## 構文

```text
actor show [キャラクターID] [状態] at [水平座標]
```

## パラメーター
| パラメーター | 必須 | 例 | 説明 |
|------|------|------|------|
| キャラクターID | はい | `alice` | キャラクターリソース識別子 |
| 状態 | はい | `angry` | 立ち絵の状態 |
| 水平座標 | はい | `2` | 水平方向の位置。分割位置を表します |

## 例
```text
# キャラクターを表示
actor show alice normal at 2
```
