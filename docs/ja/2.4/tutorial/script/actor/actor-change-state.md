# アクター状態の切り替え

## 機能説明
指定したアクターの状態を切り替えます。

## 構文
```text
actor change <キャラクター名> <新しい状態>
```

## パラメーター
| パラメーター | 必須 | 例 | 説明 |
|------|------|------|------|
| キャラクター | はい | kona | 状態を切り替えるアクター名 |
| 新しい状態 | はい | happy | 切り替え先の新しい状態 |

## 例

```text
actor change kona happy
```
