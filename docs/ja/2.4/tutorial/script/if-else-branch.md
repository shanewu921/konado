# If Else 分岐

## 機能説明

数値条件に基づいて、再生する会話内容を選択します。条件を満たさない場合は `else` 内の会話内容を再生します。`else` がない場合は、何も再生されません。
%変数はカスタム変数で、値は整数です。会話管理で設定でき、「%変数名」の形式で参照できます。`==` は判定条件で、等しいことを表します。

## 構文
```
if %変数 == 値:
    <会話>
    <会話>
else:
    <会話>
endif
```

## 例

```
if %love == 0:
    "Kona" "今日は何を飲みたい？"
else:
    "Kona" "また会えたね！"
    "Kona" "今日は何を飲みたい？"
endif
```
