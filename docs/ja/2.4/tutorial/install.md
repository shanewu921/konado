# Konado のインストール

## 必要環境

**Godot 4.6 以上**が必要です。

以下の Godot バージョンは開発者によってテスト済みで、正常に使用できます。

- 4.6
- 4.5.1
- 4.5

### プラグインのダウンロード

#### 安定版（推奨）
[Konado 最新リリースページ](https://github.com/godothub/konado/releases/latest) にアクセスし、最新安定版のプラグイン ZIP パッケージをダウンロードしてください。安定版は十分に検証されており、安定性と信頼性が高いため、本番環境での第一候補です。

中国本土のユーザーのダウンロード体験を向上させるため、以下の最適化リンクからリソースを取得することを推奨します：[Release リンク](https://atomgit.com/godothub/konado/releases)。このリンクはダウンロード速度の改善に役立ちます。

#### デイリービルド版（プレビュー版）
デイリービルド版も提供しています。最新の開発機能を試したい場合は、[Konado 全リリースページ](https://github.com/godothub/konado/releases/) にアクセスし、タイトルに `Konado Daily Build` と記載されたバージョンを探してダウンロードしてください。

開発者であり、最新の開発機能を試したい場合は、デイリービルド版をダウンロードし、使用中に見つけた問題を Issues へ報告してください。

> Caution：デイリービルド版は最新開発機能の試用のみを目的としています。十分にテストされていない新機能を含むため、未修正の Bug が存在する可能性があります。**本番環境では使用しないでください**。実際の需要に応じて慎重に選択してください。

## インストール手順

1. Godot プロジェクトディレクトリにプラグインフォルダー `addons` を作成します
2. release のプラグイン zip パッケージをダウンロードします
3. `addons` ディレクトリへ展開します
4. Godot のプロジェクト設定でプラグインを有効にし、現在のプロジェクトを再読み込みします

> Tip: 初回インストールおよびプラグイン有効化時にいくつかのエラーが表示される場合があります。無視してプロジェクトを再読み込みすれば問題ありません。
