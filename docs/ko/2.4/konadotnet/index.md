# Konado .NET API

Konado .NET API는 Konado의 C# 확장 레이어입니다. 

주요 기능:

- `KonadoAPI` 자동 로드 진입점
- `DialogueManagerAPI` 대화 관리자 제어
- Konado 대화 흐름 이벤트 구독
- `.ks` 스크립트 파싱
- `KND_Dialogue`, `KND_Shot`, `KND_DialogueChoice` 같은 GDScript 리소스 래퍼

먼저 설치 문서를 읽고, 그다음 API 참조를 확인하세요.
