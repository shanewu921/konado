# 라이선스 계약

## Konado 라이선스

`Konado`는 오픈 소스 프로젝트입니다. 우리는 더 많은 사람이 `Konado`를 사용할 수 있기를 바라기 때문에 `BSD 3-Clause` 라이선스를 선택했습니다.

Konado는 `BSD 3-Clause` 라이선스를 따릅니다. 자세한 내용은 코드 저장소의 [LICENSE](https://atomgit.com/godothub/konado) 문서를 참고해 주세요.

## 고지 정보

고지 정보란 여러분의 프로젝트에서 `Konado` 관련 정보를 명시하는 것을 말합니다. 프로젝트에서 `Konado`를 사용한다면 `BSD 3-Clause` 라이선스를 준수하여, 프로젝트 안에 본 프로젝트의 라이선스 고지 문서를 표시하거나 본 프로젝트의 고지 정보를 표시해 주세요. 예를 들어 애플리케이션이나 게임의 시작 화면에 표시할 수 있습니다.

아래는 서로 다른 상황에서 Konado 정보를 고지하는 예시입니다. 프로젝트 유형과 표시 위치에 맞게 선택하거나 수정해 사용할 수 있습니다.
