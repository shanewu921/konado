# 선택지 점프

## 기능 개요
이 기능은 **분기 기능**과 함께 사용해야 합니다. 핵심 역할은 상호작용 선택지 태그에 점프 로직을 설정하는 것입니다. 해당 선택지를 클릭하면 지정한 분기 노드로 직접 이동할 수 있습니다. 각 `choice` 문은 하나의 선택지를 생성합니다. 서로 인접한 여러 `choice` 줄은 하나의 선택지 그룹으로 병합되어 상호작용 시 함께 표시됩니다.

## 문법 규칙

```
choice "선택지 내용" -> 분기 이름
```

| 매개변수 | 필수 | 예시 | 설명 |
|------------|------|-----------------|--------------------------|
| 선택지 내용 | 예 | "커피 선택" | 상호작용 시 표시되는 선택지 텍스트 |
| 분기 이름 | 예 | coffee_choice | 이동할 대상 분기의 고유 식별자 |

## 사용 예시

각 선택지를 한 줄씩 작성:

```
choice "Green tea" -> green_tea
choice "Black tea" -> black_tea
```

한 줄에 여러 선택지를 연속해서 작성할 수도 있습니다.

```
choice "Green tea" -> green_tea "Black tea" -> black_tea
```

## 주의 사항
1. 분기 이름은 프로젝트에 정의된 분기 식별자와 완전히 일치해야 하며, 대소문자를 구분하고 공백이나 특수 기호를 포함할 수 없습니다.
2. 선택지 내용은 영어 큰따옴표로 감싸야 하며 일반 텍스트 문자를 지원합니다.
3. `->`는 선택지와 분기 이름 사이의 구분자이며 생략할 수 없습니다.
