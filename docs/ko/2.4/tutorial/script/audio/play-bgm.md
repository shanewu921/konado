# BGM 재생

## 기능 개요
특정 장면이나 상호작용 노드에서 지정한 배경 음악(BGM)을 재생하는 데 사용합니다. 장면 분위기와 감정 표현을 만드는 데 쓰이며, BGM 재생은 즉시 실행되고 이후 스크립트 실행을 막지 않습니다.

## 문법 규칙
```
play bgm <음악 이름>
```

## 매개변수 설명
| 매개변수 | 필수 여부 | 예시 | 설명 |
|------------|----------|--------------|--------------------------|
| 음악 이름 | 예 | main_theme | 재생할 BGM 파일 식별 이름 |

## 코드 예시
```
play bgm main_theme
play bgm peaceful_morning
play bgm tension_theme
```
