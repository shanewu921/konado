# 효과음 재생

## 기능 개요
특정 장면이나 상호작용 노드에서 지정한 효과음 파일을 재생해 장면 분위기와 상호작용 피드백을 강화합니다. 효과음 재생은 즉시 실행되고 이후 스크립트 실행을 막지 않습니다.

## 문법 규칙
```
play sfx <효과음 이름>
```

## 매개변수 설명
| 매개변수 | 필수 여부 | 예시 | 설명 |
|------------|----------|--------------|--------------------------|
| 효과음 이름 | 예 | door_open | 재생할 효과음 파일 식별 이름 |

## 코드 예시
```
play sfx door_open
play sfx button_click
play sfx notification
```
