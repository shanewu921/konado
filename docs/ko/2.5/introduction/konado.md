# Konado 소개

## 프로젝트 개요

Konado는 Godot Engine 기반의 비주얼 노벨 엔진 프레임워크이며, 오픈 소스 커뮤니티가 주도합니다. 우리는 개발자에게 효율적이고 편리한 비주얼 노벨 개발 경험을 제공하고, 창작자가 고품질 비주얼 노벨 작품을 빠르게 구축할 수 있도록 돕는 것을 목표로 합니다. 더 친숙하고 효율적인 도구를 추구하여 게임 창작자가 구현 세부 사항에 매몰되지 않고 스토리텔링과 예술적 표현에 집중할 수 있도록 합니다.

## 프로젝트 성과

- Konado는 Godot Engine 공식 [awesome-godot](https://github.com/godotengine/awesome-godot) 프로젝트의 "Plugins and scripts" 섹션에 추천 플러그인 중 하나로 수록되었습니다.
- Konado는 [2025 AtomGit Top 100 Open Source Projects](https://atomgit.com/GitCode/G-Star100)에 선정되었으며, [AtomGit G-Star Landscape 3.0](https://gstar-landscape.gitcode.com/?item=音视频-图形-游戏-3d--游戏引擎-unity-unreal-godot--konado) 생태계 지도에도 포함되었습니다.
