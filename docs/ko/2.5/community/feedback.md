# 제안과 피드백

`Konado` 프로젝트의 문서, 개발 절차, 구체적인 코드 등에 대한 제안이 있거나 문서와 코드에서 오류를 발견했다면 언제든지 알려 주세요.

어떤 방식으로든 참여하거나 도움을 제공하고 싶다면 [문의해 주세요](#피드백-채널). 커뮤니티가 함께 만들어 가는 오픈 소스 프로젝트인 만큼, 여러분의 기여는 `Konado`를 더 좋게 만듭니다.

## 문제 피드백

공식 저장소에 [issue를 제출](https://github.com/godothub/konado/issues)해 주세요. 주로 다음 세 가지 유형으로 나뉩니다.

- **Bug 보고**: 문제를 설명하고 재현 절차를 포함해 주세요
- **기능 제안**: Issue 영역에 요구 배경과 기대 효과를 설명해 주세요
- **문서 개선**: 개선이 필요한 부분을 Issue 영역에 설명해 주세요

## 피드백 채널

* [issue 제출](https://github.com/godothub/konado/issues) (문제를 추적하기 쉬우므로 우선 권장)

* [그룹에 참여해 피드백하기](https://pd.qq.com/g/godot)

* [게시글로 도움 요청하기](https://pd.qq.com/g/GodotHub999)
