# Konado 설치

## 필요 환경

**Godot 4.6 이상**이 필요합니다.

다음 Godot 버전은 개발자가 테스트했으며 정상적으로 사용할 수 있습니다.

- 4.6
- 4.5.1
- 4.5

### 플러그인 다운로드

#### 안정 버전(권장)
[Konado 최신 버전 페이지](https://github.com/godothub/konado/releases/latest)에서 최신 안정 버전 플러그인 ZIP 패키지를 다운로드하세요. 안정 버전은 충분히 검증되어 안정성과 신뢰성이 더 높으므로 프로덕션 환경의 첫 번째 선택입니다.

중국 본토 사용자들의 다운로드 경험을 개선하기 위해 다음 최적화 링크를 통한 리소스 다운로드를 권장합니다: [Release 링크](https://atomgit.com/godothub/konado/releases). 이 링크는 다운로드 속도 개선에 도움이 됩니다.

#### 데일리 빌드(체험 버전)
데일리 빌드도 함께 제공합니다. 최신 개발 기능을 체험하려면 [Konado 전체 릴리스 페이지](https://github.com/godothub/konado/releases/)로 이동해 제목에 `Konado Daily Build`가 표시된 버전을 찾아 다운로드하세요.

개발자이며 최신 개발 기능을 체험하고 싶다면 데일리 빌드를 다운로드하고, 사용 중 만난 문제를 Issues에 피드백해 주세요.

> Caution: 데일리 빌드는 최신 개발 기능을 체험하기 위한 용도입니다. 충분히 테스트되지 않은 새 기능을 포함하므로 수정되지 않은 Bug가 있을 수 있습니다. **프로덕션 환경에서는 사용하지 마세요**. 실제 요구에 따라 신중하게 선택해 주세요.

## 설치 단계

1. Godot 프로젝트 디렉터리에 플러그인 폴더 `addons`를 새로 만듭니다
2. release의 플러그인 zip 패키지를 다운로드합니다
3. `addons` 디렉터리에 압축을 풉니다
4. Godot 프로젝트 설정에서 플러그인을 활성화하고 현재 프로젝트를 다시 로드합니다

> Tip: 처음 설치하고 플러그인을 활성화할 때 몇 가지 오류가 나타날 수 있습니다. 무시해도 되며, 프로젝트를 다시 로드하면 됩니다.

## 개발팀 메시지

Konado 플러그인 설치가 완료되었습니다. 본격적으로 창작을 시작하기 전에 메시지 하나를 전하고 싶습니다.

> "어떤 산업에서든 상상력에 약간의 지혜가 더해지면 그것은 무엇도 막을 수 없는 무기가 된다."
>
> -- 필립 K. 딕(Philip K. Dick)  
> *Minority Report* (1956)

Konado는 아직 초기 반복 단계에 있으며, 기능이 모든 사용 시나리오를 포괄하지는 못합니다. 도구의 현재 한계가 불편을 가져올 수 있음을 잘 알고 있으며, 이에 대해 진심으로 사과드립니다. 다만 이것이 창작을 시작하는 발걸음을 지나치게 방해하지 않기를 바랍니다. 아이디어를 현실로 만드는 일은 도구의 일시적인 한계에 묶여서는 안 됩니다.

중요한 상업 게임 프로젝트를 진행 중이라면, 먼저 작은 범위에서 테스트해 Konado가 핵심 요구에 맞는지 확인하는 것을 권장합니다. 아직 사용 시나리오를 만족하지 못한다면, 프로젝트 진행을 보장하기 위해 더 성숙한 도구를 우선 선택할 수도 있습니다.

Konado를 계속 사용해 주신다면 진심으로 감사드립니다. 그것은 우리가 계속 최적화해 나가는 원동력이 됩니다.

항상 상상력과 완성도를 향한 태도를 지니고, 창작의 길에서 더 많은 가능성을 탐색하길 바랍니다.
