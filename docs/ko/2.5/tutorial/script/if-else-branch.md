# If Else 분기

## 기능 설명

숫자 조건에 따라 재생할 대화 내용을 선택합니다. 조건을 만족하지 않으면 `else` 안의 대화 내용을 재생하고, `else`가 없으면 아무 대화도 재생하지 않습니다.
%변수는 사용자 지정 변수이며 값은 정수입니다. 대화 관리에서 설정하고 `%변수명` 방식으로 참조할 수 있습니다. `==`는 같음을 의미하는 판단 조건입니다.

## 문법 구조
```
if %변수 == 값:
    <대화>
    <대화>
else:
    <대화>
endif
```

## 예시

```
if %love == 0:
    "Kona" "오늘은 뭘 마시고 싶어?"
else:
    "Kona" "또 만났네!"
    "Kona" "오늘은 뭘 마시고 싶어?"
endif
```
