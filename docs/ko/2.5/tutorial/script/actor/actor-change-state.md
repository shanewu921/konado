# 액터 상태 전환

## 기능 설명
지정한 액터의 상태를 전환합니다.

## 문법 구조
```text
actor change <캐릭터 이름> <새 상태>
```

## 매개변수 설명
| 매개변수 | 필수 | 예시 | 설명 |
|------|------|------|------|
| 캐릭터 | 예 | kona | 상태를 전환할 액터 이름 |
| 새 상태 | 예 | happy | 전환할 새 상태 |

## 예시

```text
actor change kona happy
```
