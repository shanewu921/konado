# 액터 생성

## 기능 설명

대화 장면에 액터를 생성합니다.

## 문법 구조

```text
actor show [캐릭터 ID] [상태] at [가로 좌표]
```

## 매개변수 설명
| 매개변수 | 필수 | 예시 | 설명 |
|------|------|------|------|
| 캐릭터 ID | 예 | `alice` | 캐릭터 리소스 식별자 |
| 상태 | 예 | `angry` | 스탠딩 이미지 상태 |
| 가로 좌표 | 예 | `2` | 가로 위치, 몇 번째 분할 위치인지 나타냅니다 |

## 예시
```text
# 캐릭터 표시
actor show alice normal at 2
```
