# 시그널

## 기능 설명

대화 장면 안의 사용자 지정 시그널이며, 거의 모든 작업에 사용할 수 있습니다.

## 문법 구조

```text
signal <사용자 지정 시그널 명령>
```

## 매개변수 설명
| 매개변수 | 필수 | 예시 | 설명 |
|------|------|------|------|
| 사용자 지정 시그널 명령 | 예 | 호감도 상승 | 구체적인 명령 |

## 예시
```text
# 예시
signal 호감도 상승
signal 호감도 하락
signal 골드로 호감도 설정 50
```
