# 대화 인터페이스 사용자 지정

## 소개

작품에 사용자 지정 대화 인터페이스가 필요하다면 템플릿 대화 인터페이스를 수정해 구현할 수 있습니다. 또한 Godot Engine 자체에 강력한 테마 시스템이 있으므로 사용자 지정 테마를 수정해 원하는 효과를 얻을 수도 있습니다.

다만 템플릿 대화 인터페이스를 수정한 뒤에는 Konado 플러그인을 업데이트하지 마세요. 업데이트하면 수정 사항이 덮어써집니다.

## 템플릿 파일 편집

`res://addons/konado/template/`는 대화 인터페이스 장면입니다. 이 파일을 수정해 대화 인터페이스를 사용자 지정할 수 있습니다.

일반적으로 노드의 스크립트는 수정하지 말고, 노드의 속성을 수정해 사용자 지정 효과를 구현하세요.
